# Petite Gem Quote PWA

Bản web app để mở bằng Safari/Chrome, Add to Home Screen, rồi dùng offline.

Không mở bằng iPhone Files Preview vì preview không chạy JavaScript đầy đủ.

Cách dùng trên iPhone:
1. Upload folder này lên GitHub Pages/Netlify/Vercel hoặc mở qua local server HTTPS/HTTP.
2. Mở link bằng Safari.
3. Bấm Share -> Add to Home Screen.
4. Mở icon Petite Quote trên màn hình chính. Sau lần đầu, app cache để dùng offline.
