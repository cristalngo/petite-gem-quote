# PG Designer PWA

Upload toàn bộ thư mục này lên GitHub Pages để chạy trên iPhone.

## Cách dùng trên iPhone

1. Mở link GitHub Pages trong Safari.
2. Bấm Share.
3. Chọn Add to Home Screen.
4. Mở app PG Designer từ màn hình chính.

## Workflow

- Chọn ảnh sản phẩm.
- Nhập yêu cầu khách.
- Search Product DB hoặc import sync JSON từ PG Manager.
- App tự chuẩn hóa size như `Tay 17.5mm`, tự thêm `Thân nhẫn 1.4mm`, và tự search DB theo yêu cầu/ảnh nếu DB có ảnh.
- Tick các dòng muốn gửi xưởng.
- Bấm Tạo ảnh.
- Bấm Share để gửi ảnh PNG.

## Data

- `ProductDatabase.json` là DB mặc định đi kèm package.
- Có thể import JSON mới trên iPhone. App hỗ trợ `ProductDatabase.json` hoặc mobile sync package có `products`, `orders`, `thuChiRows`.
